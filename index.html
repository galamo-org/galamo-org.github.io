Support

Galamo is an open-source Python package for comprehensive galaxy analysis, integrating machine learning and statistical methods. It provides automated tools for morphology classification, kinematics, photometry, and spectral analysis to aid astrophysical research.

The Galamo community is committed to supporting the community code of conduct.

Please remember to acknowledge and cite the use of Galamo!

What's new in Galamo 1.0

Galamo is still in its early stages, and we are continuously working to enhance its accuracy and performance.
Check out what's new in version 0.0.8 at www.galamo.org/new.

Install Galamo ¶

There are a number of ways of installing the latest release of the Galamo core package. If you normally use pip to install Python packages, you can do:

pip install galamo --upgrade

Please see the galamo installation guide for important details. This guide covers creating a Python environment, installing with pip, building from source, requirements, and testing.

Get Help¶

If you have any questions regarding using Galamo there are numerous channels for communication. Post to any one of several forums to get help from our active, helpful, and friendly community of users and developers.

Report bugs and Contribute¶

If you encounter something you believe to be a mistake, error, or bug, the best way to get it addressed is to report it on the github issue tracker. If you aren't sure if something is a bug or not, or if you don't have a Github account, feel free to ask on one of the forums. If you believe you know how to fix the problem, please consider contributing!

Support Galamo¶

If you use Galamo in your work, we would be grateful if you could include an acknowledgment in papers and/or presentations. See Acknowledging & Citing Galamo for details.

Galamo is evolving, and we are constantly working to refine its precision and performance. Your support can help us accelerate development, enhance features, and bring cutting-edge advancements to the community. If you believe in our mission, consider making a donation. Every contribution, big or small, makes a difference!

If you represent an astronomy institute and are interested in collaboration opportunities, please reach out to us at contact@galamo.org.
