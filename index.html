Galamo is an open-source Python package for comprehensive galaxy analysis, integrating machine learning and statistical methods. It provides automated tools for morphology classification, kinematics, photometry, and spectral analysis to aid astrophysical research.

    About Galamo

    Galamo is an open-source Python package designed for comprehensive galaxy analysis. It provides high-level workflows for automated morphological classification, emission-line spectral diagnostics, and low-light image enhancement, with a focus on data from large sky surveys such as SDSS and HST. Galamo is written in python including its deep learning models. Training was accelerated on an NVIDIA GeForce GTX 1650.

    Galamo includes three core modules:

    • psr for preprocessing faint galaxy images using classical image enhancement techniques

    The toolkit is built on Python libraries including Astropy, Scikit-learn, and TensorFlow, and supports both command-line use and importable modules. It includes example notebooks, batch processing support, and compatibility with standard astronomical data formats. Galamo is intended for both research and education, promoting reproducibility and efficiency in extragalactic astronomy.
