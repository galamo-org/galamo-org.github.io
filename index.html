About Galamo

Galamo is an open-source Python package designed for comprehensive galaxy analysis. It leverages the power of modern machine learning and statistical methods to provide astrophysicists and researchers with a robust toolkit for their work.

Core Features:

  • Morphology Classification: Automated classification of galaxy morphologies using deep learning models.
  • Spectral Analysis: Tools for spectral diagnostics, including BPT diagrams for AGN detection.
  • Data Integration: Seamlessly works with data from surveys like the Sloan Digital Sky Survey (SDSS) and the Hubble Space Telescope (HST).
  • Image Enhancement: Includes utilities for preprocessing and enhancing low-light astronomical images.

Technology Stack:

Galamo is built entirely in Python, relying on a foundation of well-established scientific computing libraries such as NumPy, SciPy, Astropy, and scikit-learn. For its deep learning capabilities, it utilizes TensorFlow and Keras.

Contributing to Galamo

Thank you for your interest in contributing to Galamo! We welcome contributions from everyone. By participating in this project, you agree to abide by our code of conduct.

Ways to Contribute

  • Reporting Bugs: If you find a bug, please report it on our GitHub Issues page. Include as much detail as possible, such as your operating system, Python version, Galamo version, and the steps to reproduce the error.
  • Suggesting Enhancements: Have an idea for a new feature or an improvement to an existing one? Open an issue to start a discussion.
  • Pull Requests: We welcome pull requests for bug fixes and new features. Please follow the development workflow:
    1. Fork the repository on GitHub.
    2. Create a new branch for your feature or bug fix.
    3. Make your changes and add tests for them.
    4. Ensure your code follows the project's coding style.
    5. Submit a pull request with a clear description of your changes.

Getting Started

If you are new to the project, a good place to start is by looking for issues tagged with "good first issue". Feel free to ask for help or clarification on our mailing list or the GitHub issue tracker.
