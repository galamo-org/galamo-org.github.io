Contributing to Galamo

Thank you for your interest in contributing to Galamo, an open-source Python package for galaxy analysis. Contributions are essential to the continued development and improvement of this project, and all contributions—whether in the form of code, documentation, ideas, or feedback—are greatly appreciated.

Code of Conduct

All contributors are expected to adhere to the project’s Code of Conduct.

Getting Started

  1. Fork: https://github.com/galamo-org/galamo
  2. Clone: 
    git clone https://github.com/your-username/galamo.git
cd galamo
  3. Setup: 
    python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -e .[dev]
  4. Create a branch: 
    git checkout -b your-feature-branch

Types of Contributions

  • Bug reports
  • New features
  • Code refactoring
  • Unit tests
  • Improved documentation

Coding Standards

  • Follow PEP 8
  • Use black and flake8
  • Test with pytest
black galamo/ tests/
flake8 galamo/
pytest

Before Submitting a Pull Request

  • Tests pass
  • Style is clean
  • Docs are updated
  • Related issues are referenced

Community Support

  • GitHub Issues
